#  Simple Notes API

## Описание проекта

**Simple Notes API** - это веб-приложение на Node.js для управления заметками. Позволяет создавать, просматривать, обновлять и удалять текстовые заметки через REST API.

> *Цель проекта: создать минималистичное и функциональное API для управления заметками, демонстрирующее основные принципы разработки на Node.js*

---

##  Основные возможности

-  Создание новых заметок
-  Просмотр списка всех заметок
-  Поиск заметки по ID
-  Обновление существующих заметок
-  Удаление заметок
-  Хранение данных в JSON-файле
